Indel Documentation

This page provides documentation for the Indel module.

Overview

The Indel module is designed to handle insertions and deletions in genomic sequences. It provides functions to identify, annotate, and analyze indels in DNA sequences.

Functions

  • identify_indels(sequence1, sequence2): Compares two sequences and identifies the positions of insertions and deletions.
  • annotate_indels(indel_list): Takes a list of indels and annotates them with relevant genomic information.
  • analyze_indels(indel_data): Analyzes the impact of identified indels on gene function and expression.
